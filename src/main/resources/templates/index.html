Giới thiệu ngắn

Tôi đam mê lĩnh vực công nghệ, đặc biệt chú ý đến phần cứng máy tính và điện thoại. Tôi thường xuyên cập nhật thông tin công nghệ và xây dựng các dự án thực tế để rèn kỹ năng lập trình, thiết kế hệ thống và triển khai.

Sở thích

  • Xem tin tức và bài viết công nghệ
  • Nghiên cứu, so sánh phần cứng PC và thiết bị di động
  • Phát triển ứng dụng di động và web

Dự án tiêu biểu

  • Ứng dụng quản lý chi tiêu — Flutter + SQLite. Ứng dụng giúp ghi nhận khoản thu/chi, phân loại, và xem báo cáo cơ bản.
  • Ứng dụng xem tin tức — Flutter + .NET API + MongoDB + JWT. Ứng dụng đọc tin, phân loại theo chủ đề, và có hệ thống xác thực người dùng.
  • Game 2D — Unity. Game đơn giản để thực hành logic gameplay và quản lý tài nguyên trên nền tảng Unity.
  • Website hỗ trợ xây dựng cấu hình máy tính — Spring Boot + MySQL + JWT. Công cụ giúp lựa chọn linh kiện, kiểm tra tương thích và lưu cấu hình người dùng.
